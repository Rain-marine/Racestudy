# Racestudy
This is an analysis of DICOM fMRI images. from preproccessed nifti files to final beta map images.
42 native Finnish participants were shown videos of people with 4 different races. The details of the study is not published due to privacy policies. However a subset of example data files are provided to improve the grasp of how the codes work.
This repository contains codes for processing already preprocessed fMRI images. A GLM (general linear model) has been used to produce beta maps per participant and later producing the group analysis results.
